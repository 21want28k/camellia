
# camellia-dashboard
## 简介  
可以管理和下发camellia-core的代理配置，从而可以**动态**更新代理配置  
提供了一个spring-boot-starter，可以快速的搭建dashboard  
## maven依赖
```
<dependency>
  <groupId>com.netease.nim</groupId>
  <artifactId>camellia-dashboard-spring-boot-starter</artifactId>
  <version>a.b.c</version>
</dependency>
```
